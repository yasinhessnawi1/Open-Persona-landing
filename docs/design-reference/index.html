The problem

AI forgets.
Personas drift.

A stateless model rebuilds "who it is" from scratch every turn. Keep talking and the character erodes — facts contradict, the voice wanders, the constraints slip. Open Persona holds the line.

Typed memory

Four stores.
One mind.

Memory isn't a blob of chat history. Each persona has four structured, versioned, typed stores — plus enforced constraints. Every one is queryable, auditable, and the same whether the turn came by voice or text.

ID

identity

Who they are — name, role, the spine of the character. Immutable at runtime.

read-only · validated on author

SF

self_facts

What they've learned about themselves over time — capabilities, specialisations, limits.

append-only · versioned · rollback

WV

worldview

Their views and stances — the opinions that make the voice consistent instead of mushy.

typed · epistemic markers

EP

episodic

Everything from past conversations — retrieved, summarised, compacted. The long memory.

pgvector + Chroma · summarise & compact

+ enforced constraints — the things a persona will not do

Real-time voice

Say it out loud.
It remembers when you type.

Talk to your persona hands-free; it answers in its own synthesized voice, sub-second. Voice and text aren't two products bolted together — they share the same memory, the same constraints, the same identity. Speak a fact now; type next week and it's still there.

▮ Voice · spoken"Remember I'm contesting the deposit, not the rent."
⌨ Text · 6 days laterWhat was my case about again?
▮ Same persona · same memory"The deposit deduction for the smoke damage — you weren't contesting the rent."

Capabilities

A persona that can
actually do things.

Memory makes it consistent. Tools, skills, and rich output make it useful — all rendered inline, all in character.

Tools

Web search, calculator, currency, image generation, code execution — and any MCP server in the wild.

Skills

Document generation — docx, pdf, pptx, xlsx, markdown — produced on demand from a conversation.

Rich output

Images, Mermaid & Graphviz diagrams, and files render inline in the conversation, not as links.
